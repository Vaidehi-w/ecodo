# EcoDo AI 🌍
**Track your impact, save the planet.**

EcoDo AI is a React-based sustainability dashboard that uses **Google Gemini 1.5 Flash** to analyze daily habits and provide scientifically-grounded environmental impact data.

## ✨ Features
- **AI Impact Analysis:** Real-time feedback on CO2, Water, and Energy savings.
- **Dynamic Visualization:** Interactive Pie Charts showing your daily contribution.
- **Eco-Streak:** Gamified tracking to keep you motivated.
- **AI Assistant:** A casual chatbot to answer all your "green" questions.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **AI:** Google Gemini API
- **Charts:** Recharts
- **Styling:** Custom CSS (Nature-inspired)

## 🚀 Setup
1. Clone the repo
2. Run `npm install`
3. Add your `VITE_GEMINI_API_KEY` to a `.env` file
4. Run `npm start`
